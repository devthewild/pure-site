커스텀하기

필요한 모듈만 선택

묶음

묶음들에는 Pure의 모든 모듈이 들어있다. 2가지 묶음이 있는데, 하나는 보통(반응형)이고, 하나는 비-반응형이다. 프로젝트에 맞는 것을 고르면 된다.

이름 CSS URL 크기 (압축됨)
반응형 몪음 http://yui.yahooapis.com/pure/0.2.0/pure-min.css 4.2KB
비-반응형 묶음 http://yui.yahooapis.com/pure/0.2.0/pure-nr-min.css 4.0KB

개별 모듈들

모듈들을 개별적으로 가져올 수도 있다. 야후 CDN은 콤보 선택을 제공하는데, 요청에 따라 하나의 CSS 파일을 받을 수 있다. 아래 예시에서는 기본, 그리드, 폼만 포함된 CSS를 받을 수 있는 콤보다. 

<link rel="stylesheet" href="http://yui.yahooapis.com/combo?pure/0.2.0/base-min.css&pure/0.2.0/grids-min.css&pure/0.2.0/forms-min.css">
이름 CSS URL
기본 http://yui.yahooapis.com/pure/0.2.0/base-min.css
버튼 http://yui.yahooapis.com/pure/0.2.0/buttons-min.css
폼 (반응형) http://yui.yahooapis.com/pure/0.2.0/forms-min.css
폼 (비-반응형) http://yui.yahooapis.com/pure/0.2.0/forms-nr-min.css
그리드 (반응형) http://yui.yahooapis.com/pure/0.2.0/grids-min.css
그리드 (비-반응형형) http://yui.yahooapis.com/pure/0.2.0/grids-nr-min.css
메뉴 (반응형) http://yui.yahooapis.com/pure/0.2.0/menus-min.css
메뉴 (비-반응형형) http://yui.yahooapis.com/pure/0.2.0/menus-nr-min.css
테이블 http://yui.yahooapis.com/pure/0.2.0/tables-min.css

스킨 만들기

Pure는 최소화된 스타일이라 자신의 CSS로 커스텀할 수 있다. YUI Skin Builder는 테마를 만들어주는 좋은 툴이다. 클릭 한번으로 Skine Builder는 Pure 테마에 필요한 모든 CSS를 생성해준다.
